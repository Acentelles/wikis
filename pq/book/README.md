# Isogeny-Based Cryptography Notes

Personal study notes on isogeny-based cryptography, primarily focused on the
constructions and proof systems built from group-action assumptions.

The book runs on [mdBook](https://rust-lang.github.io/mdBook/). Serve locally with:

```sh
cd book && mdbook serve
```

## Deps

- `cargo install mdbook`
- `cargo install mdbook-katex` — LaTeX is rendered at compile time with [mdbook-katex](https://github.com/lzanini/mdbook-katex).
